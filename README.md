# 🌐 Portfolio Website – R. Abida

A modern, responsive **AI & Data Science portfolio website** showcasing my skills, projects, resume, and professional journey. This portfolio is designed with a clean dark theme, smooth animations, and an interactive user interface to present my work effectively to recruiters and collaborators.

---

## 🚀 Live Preview

> abida-portfolio-website.vercel.app

---

## 📌 About the Portfolio

This portfolio website serves as a central hub for my academic and professional profile as a **B.Tech student specializing in Artificial Intelligence & Data Science**. It highlights my technical expertise, hands-on projects, internships, certifications, and a downloadable resume.

The design focuses on a **modern dark UI**, readability, and smooth user experience, making it suitable for professional and recruitment purposes.

---

## 🧠 Features

* ✨ Modern dark-themed UI with cyan accent colors
* 🎯 Interactive hero section with typing animation
* 🧩 Skills section with animated progress bars
* 📂 Projects showcase with detailed descriptions
* 📄 Downloadable resume (HTML & PDF)
* 🌌 Particle background animation
* 📱 Fully responsive design (mobile & desktop)

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **UI & Effects:** CSS Animations, Particles.js
* **Fonts:** Google Fonts (Poppins)
* **Deployment:** GitHub Pages (recommended)
